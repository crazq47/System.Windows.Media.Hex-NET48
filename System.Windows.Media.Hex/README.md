# System.Windows.Media.Hex???? System.Windows.Media.Hex  is a .NET library for Windows Presentation Foundation (WPF) that provides enhanced functionality for working with hexadecimal color codes.This project complements and extends the functionality of the standard System.Windows.Media.Color structure, fixing some shortcomings and adding new features.???? System.Windows.Media.Hex - це бібліотека родзроблена для Windows Presentation і дозволяє повноцінно керувати шістнадцятирядковими записами кольорів.Проект реалізує наявні можливості з її сестринскої структури System.Windows.Media.Color випрвляє нефункціонуючий код та розширює її функіонал.???? Бібліотека складається з трьох основних частин: базова структура HexColor, класу HexColors, який зберігає у собі готові кольори (аналог Colors), класу HexCode, який реалізує засобами взаємодії з іншими представленнями.???? Навідміну від структури Color, HexColor набагато гнучкіший у плані реалізації коду, приклад:Color color1 = new Color();color1 = Color.FromRGB(255, 0, 255);HexColor hex1 = color1; // Color записColor color2 = hex1; // обернена конвертація// взаємодія з іншими стркутурамиHexColor hex2 = 0xFF00FF; // uint записHexColor hex3 = "#FF00FF"; // string записЗавдяки невизначеним операторам, HexColor дуже добре конвертується у Color, uint та string і навпаки, здебільшого можна навіть не вказувати оператори та не використовувати методи конвертації, наприклад.Foreground = hex2; // конвертація одразу у Color.Чому ця бібліотека краще за System.Windows.Media.Color?- Реалізує не працюючий код Color.- Повторює весь наявний функіонал Color.- Розширує методи та оператори конвертації та взаємодії.- Надає можливість керувати окремими діапазонами кольору.- Дозволяє керувати записом власного коду.Структура HexColor реалізує код для керування записами коду, це включає в себе підтримку скороченого запиу (#FF00FF -> #F0F і навпаки), методи устаткування діапазонів та багао іншого.
